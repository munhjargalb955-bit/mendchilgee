# Романтик мэндчилгээний вебсайт

Энэ бол утсанд зориулж хийсэн, монгол хэлтэй романтик мэндчилгээний вебсайт юм. Бэлэг нээх хэсэг, хөдөлдөг `ҮГҮЙ` товч, баярын эффект, шударч үзэх зургийн цомог, хөгжим болон дахин эхлүүлэх товчтой.

## Ажиллуулах

1. ZIP файлыг бүрэн задлана.
2. `index.html` файлыг Chrome эсвэл Safari хөтөчөөр нээнэ.
3. `Бэлгээ нээх` товчийг дарахад хөгжим тоглож эхэлнэ.

Зураг болон хөгжмийн файлууд ZIP дотор бэлэн орсон. Файлын нэр болон хавтасны бүтцийг өөрчилж болохгүй.

## Хавтасны бүтэц

```text
mongolian-romantic-greeting/
├── index.html
├── style.css
├── script.js
├── README.md
├── images/
│   ├── photo1.jpg
│   ├── photo2.jpg
│   ├── photo3.jpg
│   └── photo4.jpg
└── audio/
    └── music.mp3
```

## GitHub Pages дээр байршуулах

1. Шинэ GitHub repository үүсгэнэ.
2. Бүх файл болон хавтсыг бүтцээр нь upload хийнэ.
3. `Settings → Pages` хэсэг рүү орно.
4. `Deploy from a branch` сонгоно.
5. `main` branch болон `/ (root)` сонгоод хадгална.

## Vercel дээр байршуулах

1. Төслөө GitHub руу upload хийнэ.
2. Vercel дээр `Add New → Project` сонгоно.
3. GitHub repository-гоо холбоно.
4. Framework-ийг `Other` гэж сонгоно.
5. Build command оруулахгүйгээр `Deploy` дарна.
